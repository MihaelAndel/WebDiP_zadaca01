Android Q beta

Android Q je najnovija, još u razvoju, verzija Android operacijskog sustava. Nedavno je izašla takozvana "Developer preview" verzija Android Q sustava za Google-ov Pixel pametni telefon. Priča se da će Android Q beta verzija biti dostupna na mnogo više uređaja nego što je to bio slučaj s prijašnjom verzijom Android sutava - Android Pie.

Datum izlaska Microsoft Visual Studio 2019

Microsoft je nedavno najavio datum za svoj "Visual Studio 2019 Launch event". Naime, događaj će se održati na webu 2. travnja 2019. Microsoft Visual Studio 2019 sadržavat će mnogo novih funkcionalnosti od kojih je najinteresantnija AI pomoć u pisanju koda. Na navedenom događaju će se također prikazati ostale nove stvari u razvoju, poput nove verzije .NET Core okvira.
